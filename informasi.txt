===========================================================
INFORMASI PROYEK: AI CODE ANALYZER (WEB 2)
===========================================================

1. TENTANG APLIKASI
Web 2 (AI Code Analyzer) adalah aplikasi berbasis kecerdasan buatan (Groq API) yang dirancang untuk menganalisis, mendeteksi bug, me-review, dan mengonversi kode pemrograman secara otomatis. Aplikasi ini dibangun sebagai bagian dari pemenuhan tugas mata kuliah NLP Kasus 4: Code Generation. Fokus utamanya adalah membantu developer memahami dan memperbaiki kode dengan cepat dan presisi.

2. TEKNOLOGI UTAMA
- Next.js 16 (App Router)
- TypeScript & Tailwind CSS v4
- Framer Motion (Transisi dan UI animasi)
- react-markdown & react-syntax-highlighter (Rendering Markdown & sintaks kode)
- API AI: Groq (Model llama-3.3-70b-versatile)

3. DAFTAR 5 USE CASE / FITUR UTAMA
   A. Code Converter
      - Konversi kode dari satu bahasa ke bahasa lain.
      - Mendukung 10 bahasa: Python, JavaScript, TypeScript, Java, Go, PHP, C++, Rust, Swift, Kotlin.
      - Output berupa pure code yang dilengkapi warna highlight syntax (tanpa markdown).
   
   B. Code Explainer
      - Menerima blok kode dan mengembalikan penjelasan baris per baris.
      - Sangat berguna untuk memahami kode asing yang rumit.
      - Output menggunakan format Markdown rapi dalam Bahasa Indonesia.

   C. Bug Finder (Baru)
      - Spesialis pendeteksi kesalahan dan bug pada kode.
      - Detail output: (1) letak bug, (2) nama bug, (3) alasan mengapa itu salah, dan (4) kode perbaikannya.
      - Jika kode sudah bersih (tidak ada bug), AI akan menjawab tegas: "✅ Code looks good! Tidak ditemukan bug pada kode ini."
      
   D. Code Review
      - Sistem penilaian kode selayaknya code reviewer profesional.
      - Menghasilkan: Skor 1-10, list poin kelebihan, list poin kelemahan, dan rekomendasi konkrit (best practices).

   E. Security Audit
      - Analisis keamanan sistem (vulnerabilities scanning).
      - Menemukan celah (seperti SQL Injection, XSS) lengkap dengan tingkat bahaya (Critical, High, dsb) serta langkah perbaikannya.

4. USER EXPERIENCE (UX) HIGHLIGHTS
- Dark Mode / Glassmorphism: Desain gelap bersih dan minim distraksi mata.
- Validasi Input: Tombol hanya aktif jika panjang kode minimal 5 karakter.
- Shortcut Pintar: CTRL + ENTER untuk menjalankan analisis.
- Copy Output: Memiliki dua jenis copy: (1) Copy global untuk menyalin seluruh narasi dan hasil analisis, (2) Copy spesifik (*hover-only*) pada setiap blok kode Markdown untuk menyalin kodenya saja. Keduanya dilengkapi indikator ("Copied!").
- Timer Eksekusi: Informasi berupa seberapa cepat AI berhasil membalas prompt ("Generated in 2.1s").
- Render Elegan: Semua output menggunakan layout yang sudah dirapikan oleh tailwindcss/typography dan otomatis dibungkus sehingga tidak meluap ke samping (horizontal scrolling).

5. CATATAN KONFIGURASI
Semua permintaan / prompt untuk AI berpusat di backend (`app/api/analyze/route.ts`). Kunci rahasia tersimpan di file `.env.local` sehingga kerahasiaan Groq API Key aman dan tidak terekspos di browser client.
